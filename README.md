
# دوران

## ✨ ویژگی‌های برنامه‌ریزی‌شده

- [ ] تبدیل تاریخ بین تقویم‌های مختلف (شمسی، میلادی، قمری؟)
- [ ] نمایش رخدادها و وقایع بر اساس دسته‌بندی‌های مرسوم با قابلیت فعال کردن به صورت انتخابی:
    - باستانی
    - رسمی
    - جهانی
    - و ...
- [ ] تنظیمات تم
